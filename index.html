IIITS Telugu Handwriting Database

Introduction

The IIITS-THDB is the IIITS-Telugu Handwriting database. It is a collection of 450 Telugu handwriting samples written by 150 writers. The database was created as part of the work done for [1]. This database may be used for non-commercial research and educational purpose only. If you publish material based on this database, please include a reference to the paper [1].

Telugu Script Overview

The telugu writing system is an Abugida or a segmental writing system in which each unit of the script consists of both a consonant and a vowel. This is different from a full alphabet script such as English, where both vowels and consonants are given equal status. Although vowels in Telugu are written independently wherever required, their use in the independent form is usually limited. More than 800 million users around the globe use Telugu and it is one of the most prominent languages in India and Asia. Characteristics of the script are quite unique and at a glance, one can see that the characters and symbols in the language are rounded in shape. The Telugu script comprises of smooth rounded curves that are joined together to create characters. The script does not include any vertical strokes or sharp junctions at regions of change in trajectory. The change in trajectory in Telugu writing is smoothened out by elongating the curve at the point of change. Spatial positioning of strokes is also an important factor while writing Telugu, as is the case with most Indian languages. A more detailed analysis of various Indian scripts is done in [1].

IIITS - THDB

IIITS-THDB is the IIITS-Telugu Handwriting Database. As part of the work done for this paper, we collected samples of offline Telugu handwriting for writer identification. To the best of the author’s knowledge, no standard dataset for handwritten offline text exists for the Telugu language. For the purpose of writer identification, we collected a set of 450 handwritten Telugu pages from a total of 150 writers, with 3 pages written by each writer. A total of 20 Telugu paragraphs were collected from textbooks, ancient scripts, newspapers and online articles to capture diverse combinations of slangs, wordings and writing styles. Each writer was given 3 of the 20 paragraphs at random to write, ensuring that no two pages written by a writer have the same content. Each writer was also given a blue color ball point pen for writing to minimise noise caused by external factors. Writers were mostly students with around 75 writers in the 15-17 years age group, 65 writers in the 19-25 years age group and the remaining writers greater than 25 years of age. There were almost equal amounts of male and female writers in the considered group.

The collected handwritten documents were scanned at 300 dpi, converted to grayscale, binarized using Otsu's algorithm [2] and annotated manually. The handwritten part of each document was extracted using projection profiles and bounding box detection. Both the extracted handwritten parts and the original collected documents are publicly available to use for research in handwriting or any other related fields.

Naming convention

There are 2 files available for download.
The file documents.zip contains the raw scanned grayscale documents. Each of these documents have been scanned in batches and the naming conventions are batch wise. The first letter enumerates the batch in which the document was scanned and the numbers following the underscore denote the document number.

The file segment.zip contains the extracted handwritten part of each document. Each file in segment.zip has the same name as the coresponding scanned document in documents.zip.

Apart from this, each *.zip file contains a writerids.csv file that contains the annotations of the scanned images in the zip file. The writerids.csv file contains two columns, the first column denotes the filename, the second denotes the ID of the writer. Documents with same ID have been written by the same writer.
